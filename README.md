# suncompass
Website which calculates the times of sunrise, sunset, dawn, dusk, solar noon, and solar midnight; the subsolar point (point on Earth at which the sun is directly overhead), solar noon, and solar midnight.

Prereqs: Node.js (LTS recommended)

## Setup instructions
1) Clone the GitHub repository: `git clone https://github.com/crosscountrycoder/suncompass.git`
2) Move into the directory: `cd suncompass`.
3) While in the repo's root directory, run `npm install` to download dependencies.

## Running scripts
To run a file in the scripts folder, run `npx ts-node scripts/<file-name.ts>` with the relevant command-line arguments. For a list of arguments, see the README.md file within the scripts folder.

## Deploying website to localhost
To deploy the website to localhost, run `npm run dev`.
